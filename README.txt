# Transcrição de Áudio com IA - Versão 1.5

Este aplicativo permite transcrever áudios automaticamente usando inteligência artificial (modelo Whisper).

---

## 🧾 Como usar

1. **Execute o arquivo `TranscricaoBotRegis.exe`**
   - Dê dois cliques no ícone ou abra pelo terminal.

2. **Escolha um arquivo de áudio**
   - O programa aceita formatos como `.mp3`, `.wav`, `.m4a` e outros compatíveis com o ffmpeg.

3. **Aguarde a transcrição**
   - O tempo depende do tamanho do áudio.

4. **Salve o texto transcrito**
   - O aplicativo exibirá a transcrição e permitirá salvar como `.txt`.

---

## ⚙ Requisitos

- O programa já vem com os arquivos necessários.
- **Você deve manter a pasta `ffmpeg` no mesmo local do executável**, pois ela contém os binários do ffmpeg usados para processar os áudios.

---

## 🛠️ Em caso de erro

- Certifique-se de que a pasta `ffmpeg` e `whisper_models` estão no mesmo diretório do `Transcricao.exe`.
- Se aparecer alguma janela preta com mensagem de erro, envie o conteúdo para o desenvolvedor.
